# simulador-financiamento
Projeto Spring Boot (Java 21) - Simulador de Financiamento imobiliário (PRICE e SAC)

## Como rodar
Pré-requisitos: Java 21 instalado e Maven.

1. Build e rodar via Maven:
   - `mvn clean package`
   - `mvn spring-boot:run`

2. Ou executar o jar gerado:
   - `mvn clean package`
   - `java -jar target/simulador-financiamento-0.0.1-SNAPSHOT.jar`

A aplicação ficará disponível em: http://localhost:8080
